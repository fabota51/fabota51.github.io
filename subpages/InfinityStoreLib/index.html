InfinityStoreLib

Python Library Discord AES-Verschlüsselung Open Source 2025

InfinityStoreLib ist eine Python-Bibliothek, die Discord-Webhooks als verschlüsselten Cloud-Speicher nutzbar macht – direkt aus eigenem Code heraus. Entstanden aus den Erkenntnissen von InfinityStorage, wurde der Kern als eigenständige, wiederverwendbare Library herausgelöst. Mit vier einfachen Funktionen lassen sich Dateien und Python-Objekte sicher hochladen und wieder abrufen.

01 Warum eine Library?

InfinityStorage war eine vollständige Desktop-App. Die eigentliche Logik – verschlüsseln, aufteilen, per Webhook hochladen – ist aber unabhängig von der GUI nützlich. InfinityStoreLib extrahiert genau diesen Kern: eine saubere Python-API, die sich in jedes Projekt einbinden lässt, ohne die ganze Oberfläche mitzuschleppen.

02 Die vier Kernfunktionen

Die API besteht aus zwei Paaren: eines für Dateien, eines für Python-Objekte. Jeder Upload gibt einen SHA-512-Hash zurück – das ist der eindeutige Schlüssel zum späteren Abruf.

from InfinityStoreLib import upload_location, download_location, upload_data, download_data
# Datei hochladen → gibt SHA-512-Hash zurück
file_hash = upload_location("./meine_datei.zip")
# Datei wieder herunterladen
download_location(file_hash, "./ausgabe.zip")
# Python-Objekt speichern
data_hash = upload_data({"user": "fabota51", "score": 42})
# Python-Objekt abrufen
obj = download_data(data_hash)

03 Wie es intern funktioniert

Unter der Haube läuft dieselbe Pipeline wie bei InfinityStorage: Daten werden mit pyAesCrypt (AES-256) verschlüsselt, in Blöcke von 5–7 MB aufgeteilt und über mehrere Discord-Webhooks verteilt. Fake-Dateinamen (via Faker) verschleiern die Inhalte. Ein lokaler Index speichert, welche Chunks auf welchen Webhooks liegen, damit der Download die Stücke in der richtigen Reihenfolge zusammensetzen kann. Der zurückgegebene SHA-512-Hash identifiziert die Datei eindeutig im Index.

04 Konfiguration

Webhook-URLs werden entweder direkt im Code übergeben oder über eine YAML-Konfigurationsdatei eingelesen. Die Library ist MIT-lizenziert und auf GitHub verfügbar.

05 Fazit

InfinityStoreLib ist der logische nächste Schritt nach InfinityStorage: Die Kernidee – Discord als Speicher-Backend – als wiederverwendbares Werkzeug für andere Python-Projekte. Vier Funktionen, null Server, volle Verschlüsselung.
